
## Flux vidéo avec Latence

1. Sur le site web, vérifiez que "Flux externe" est bien la source pour le flux vidéo, si ce n'est pas le cas, alors un problème est arrivé coté webcam/robot, qui vous a fait passé sur un fallback moins efficace (webrtc).

2. Votre connexion peut simplement être mauvaise, essayez de vous rapprocher du robot pour améliorer la connexion wifi.

3. Si cela ne suffit pas, re-configurer les paramètres du flux vidéo pour le rendre moins lourd et donc réduire la latence (balance qualité/latence). VOIR Configuration générale/ Réglages caméra.
